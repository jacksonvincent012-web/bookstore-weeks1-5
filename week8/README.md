# Week 8 - Responsive Web Design & Mobile-First Development

This week focuses on responsive design principles using HTML and CSS.

## Contents

| File | Description |
|------|-------------|
| `profile.html` | Responsive profile page with Flexbox layout |
| `products.html` | Product showcase page with CSS Grid (4 products) |
| `css/style.css` | Mobile-first styles with media query breakpoints |
| `images/` | Place your profile and product images here |

## Features

- **Mobile-first** CSS with two breakpoints (600px tablet, 900px desktop)
- **Flexbox** layout for the profile card
- **CSS Grid** for the product showcase (1 col mobile, 2 col tablet, 4 col desktop)
- **Responsive images** using `max-width: 100%` and `height: auto`
- **Viewport meta tag** for proper scaling on mobile devices

## Images Needed

Place your images in `images/`:
- `profile.jpg` — your profile picture (square crop works best)
- `book1.jpg` through `book4.jpg` — product images (landscape or square)
